## Setup

#1. homebrew 설치
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

#2. imagesnap 설치
brew install imagesnap

#3. imagemagick  설치
brew install imagemagick

#4. ffmpeg 설치
brew install ffmpeg

#5. rename 설치  - 아직 소개하지 않은 rename 도구
brew install rename

## Run
./capture.sh
